Welcome to **Holt**, OpsMill's secondary GitHub organisation. If you've been invited here, it's almost certainly for a project or collaboration with the OpsMill team.

## Why "Holt"?

🦦 Otters live in holts, the place where the community gathers, regulars and visitors alike. (OpsMill has a recurring otter habit. We lean into it.)

## What lives here

The main organisation at [github.com/opsmill](https://github.com/opsmill) is home to Infrahub itself, the SDK, integrations, and everything we maintain as supported open-source. Holt is where shared, time-bounded work happens alongside that.

## About OpsMill

OpsMill builds [Infrahub](https://github.com/opsmill/infrahub), the open-source data and automation platform for network, data center, and cloud infrastructure. Infrahub is the graph-based, schema-first source of truth that automation tools and AI agents both depend on.
